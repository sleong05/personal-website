Treasure Chest (Project 2)

Samuel Leong • September 30, 2025 • CS 417 • Computer Animation

Reference & Planning

Reference Images

I chose a realistic, iron-banded wooden chest as my primary reference that I had previosuly used in an escape room.

All reference photos are of real chests (non-CG).

Modeling Plan

  • Block out base with polygonal primitives (box + cylindrical lid arch).
  • Use Extrude, Bevel, Insert Edge Loop, and Poke for detail breaks.
  • Add iron bands, hinges, hasp/lock plate as separate meshes.
  • Non-destructive smoothing groups; keep quads where possible.

Wireframe Views

Orthographic and perspective wireframes to show topology and supporting edges.

Smooth-Shaded Renders

Key angles of the closed and open chest in smooth shading (with and without textures).

Shading & Materials

Wood uses a base color with subtle roughness variation and a normal map; metal bands use a slightly roughened metal workflow shader with edge wear. Interior lined with darker wood for contrast.

Chest Open/Closed + Treasure

Both states are shown with coins, gems, and a goblet placed inside to satisfy the project brief.

Attributions: coins and gem meshes downloaded from 3drender.com (credited on this page); goblet modeled by me.

Enhancements

  • Custom hinge and latch assembly with beveled accents.
  • Procedural roughness breakup for wood (adds aged look).
  • Star-field environment light with subtle rim to separate silhouette.

Process Notes

Modeling: Polygonal workflow in Maya using Extrude, Bevel, Multi-Cut, and Poke; kept clean quads around curvature for stable smoothing.

UVs: Unwrapped with shell straightening for wood grain alignment; stacked mirrored shells where appropriate to optimize texel density.

Lighting/Render: Key-fill-rim setup with area lights; added environment for subtle reflections; rendered sequence for the turntables.

Credits: Any downloaded treasure assets are listed above with sources; everything else modeled by me.
